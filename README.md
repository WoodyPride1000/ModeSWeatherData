## 1. 概要

本プログラムは、ADS-B（Automatic Dependent Surveillance-Broadcast）受信データを元に、航空機の気象観測値（気温・風向・風速）を算出し、CSVファイルに出力するバッチ処理である。

ラズベリーパイ等で収集したモードSデータファイルを入力とし、特定空域内の有効データのみを対象として気象パラメータを計算する。

## 7. 注意事項・制約

磁気偏角（7.64度）はハードコードされており、時代・地域により変化する。定期的な見直しを推奨する。

出力数値は double のまま出力されるため、小数点以下の桁数が多くなる場合がある。必要に応じて String.format() による丸め処理を追加することを推奨する。

入力ファイルのインデックス[9]は未使用。入力仕様に変更がある場合は確認が必要。

文字コードはシステムデフォルトに依存するため、環境によっては文字化けが発生する可能性がある。

ネットワーク共有（\\192.168.1.zz\...）のIPアドレスは実環境に合わせて変更すること。

## 8. 謝辞

本プログラムの開発にあたり、石川 宏 博士（ナチュラル研究所）が公開された研究成果を参考にさせていただきました。

同資料より特に以下の点について多大な知見を得ました。

気温算出の物理モデル（マッハ数・真対気速度から音速を逆算し外気温を求める手法）および定数値（比熱比 k = 1.403、分子量 M = 28.966×10⁻³、気体定数 R = 8.314472）

風向・風速の算出方法（対地速度ベクトルと対気速度ベクトルの差分から風ベクトルを求める手法）

磁気偏角補正の考え方および補正値（7.64度）の採用

入力データ形式（ModeSデータのフィールド定義、ICAOアドレスによるデータ名寄せ処理の概念）

対象空域の設定（緯度34.75〜36.5度、経度137.5〜140.5度）

また、PyModeS ライブラリ（Junzi Sun, デルフト工科大学）についても、同資料を通じてその活用方法を参照しました。

本プログラムのオリジナルの設計コンセプトは石川博士に帰属します。本設計書に記載のJava実装はそれをもとに再構成・改良したものです。

石川 宏 博士（1942〜2025）

工学博士。早稲田大学大学院修了後、日本電信電話公社（現NTT）に入社。

研究所長・ネットワーク部長・役員を歴任。電子情報通信学会名誉員。

退職後も気象観測・電子工作への探究心を持ち続け、ModeSを用いた

高層気象観測システムを独力で開発・公開された。

令和7年8月12日、83歳にて御逝去。謹んで哀悼の意を表します。
